📄 Gerador Profissional de Políticas de Privacidade & LGPD/GDPR
Este é um script de ferramenta de utilidade pública e jurídica, desenvolvido em HTML5, CSS3 e JavaScript Puro, ideal para agências de web design, freelancers e proprietários de negócios que precisam de conformidade legal rápida e profissional.

✨ Funcionalidades Premium
Multilíngue Instantâneo: Alterna entre Português (LGPD) e Inglês (GDPR) com um único clique.

Exportação Inteligente:

PDF: Gera o documento pronto para impressão ou envio.

HTML: Fornece o código formatado para inserir diretamente em sites.

Texto: Cópia limpa para documentos de texto.

Modo Dark/Light: Interface moderna com troca de tema dinâmica.

Barra de Progresso: Feedback visual enquanto o usuário preenche os dados.

Link de Auto-Preenchimento: Gera um link exclusivo com os dados já preenchidos para fácil partilha ou consulta futura.

Máscaras Automáticas: Formatação inteligente de CPF/CNPJ em tempo real.

Privacidade: Não utiliza base de dados; todo o processamento é feito no navegador do cliente (Client-side).

🚀 Como Utilizar
Baixe o arquivo index.html

Abra o arquivo em qualquer navegador (Chrome, Edge, Firefox, Safari).

Preencha os campos: Razão Social, Documento, E-mail e Dados Coletados.

Escolha o idioma desejado.

Clique em Copiar Texto ou Baixar PDF.

🛠️ Tecnologias Utilizadas
Google Fonts (Inter): Para uma tipografia moderna e legível.

Font Awesome 6: Ícones profissionais.

jsPDF: Biblioteca para conversão de texto em arquivos PDF de alta qualidade.

CSS Variables: Para gestão eficiente de temas (Dark/Light).
